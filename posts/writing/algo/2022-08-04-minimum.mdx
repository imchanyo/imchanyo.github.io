## 세 수 중 최솟값

Q. `100이하의 자연수 A, B, C`를 입력받아 세 수 중 가장 작은 값을 출력하는 프로그램을 작성하세요.
(정렬을 사용하면 안됩니다)

▣ 입력설명

- 첫 번째 줄에 100이하의 세 자연수가 입력된다.

▣ 출력설명

- 첫 번째 줄에 가장 작은 수를 출력한다.

## 내가 작성한 코드 - 1

```tsx
function solution(a, b, c) {
  let answer;
  const d = a > b ? b : a;
  answer = d < c ? d : c;
  return answer;
}

console.log(solution(10, 20, 30));
```

## 내가 작성한 코드 - 2

```tsx
function solution(a, b, c) {
  let answer;
  if (a < b) {
    return (answer = a);
  } else if (a > b) {
    return (answer = b);
  } else if (answer > c) {
    return (answer = c);
  }
  return answer;
}
```

## 답안지에 적힌 코드

```tsx
function solution(a, b, c) {
  let answer;
  if (a < b) answer = a;
  else answer = b;
  if (c < answer) answer = c;
  return answer;
}
```
