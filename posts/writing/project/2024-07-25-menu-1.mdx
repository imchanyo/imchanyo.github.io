## 화면구조
![스크린샷 2023-12-26 오후 12.37.04](/img/project/menu.png)

 
위의 사진에 있는 UI는 일반적으로 사용되고 있는 관리자 시스템화면입니다. 
전체적인 구조는 다음과 같습니다.
`Side / Main Content` 영역으로 섹션이 2개로 나누어집니다. 물론 해더도 있지만 위의화면 고려하겠습니다.
그리고 메뉴관리에서 Side에 있는 메뉴들을 등록,수정,삭제를 할 수 있는 관리페이지입니다.


## 순서도
<img src="/img/project/menu-3.png" width="500" />

대략적인 순서도는 위와같습니다.
사용자가 로그인 접속을 하여 페이지에 접속하게되면 사용자의 권한그룹을 검증하여 메뉴리스트 데이터를 받습니다. 메뉴리스트 데이터는 세션스토리지에 담게됩니다. 세션스토리지에 보관하는 이유는 다음과 같습니다.
- 메뉴리스트는 개발자와 사전협의 되지않고 추가를 할 수 없기때문에 페이지 이동할 때 마다 API 호출이 불필요 합니다.
